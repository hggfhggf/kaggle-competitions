# Kaggle Competitions

This repository contains my solutions for Kaggle competitions, AI olympiads, and machine learning tournaments.

The goal of this repository is to collect practical notebooks, experiments, and final solutions for different computer vision, NLP, tabular, and optimization tasks.

## Repository Contents

| File | Description |
|---|---|
| `CEOAI CorruptDecoder Solution.ipynb` | Solution notebook for the CEOAI CorruptDecoder task |
| `NEOAI-day2-cv-0-977-solution.ipynb` | Computer vision solution for NEOAI Day 2 |
| `SimplePuzzleSolver.ipynb` | Puzzle-solving approach using image matching / geometry |
| `ceoai-practice-round-1-solution-for-star-observato.ipynb` | Practice round solution for CEOAI Star Observatory task |
